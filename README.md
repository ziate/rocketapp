# نظام إدارة توصيل الطلبات

مشروع لوحة تحكم متكاملة لإدارة الطلبات والعملاء والمناديب والمحافظات والمناطق وأنواع الطلبات.

## المميزات
- لوحة تحكم عربية بالكامل مع قائمة جانبية واضحة.
- إدارة المحافظات والمناطق مع أسعار التوصيل الافتراضية.
- إدارة العملاء مع إمكانية تخصيص أسعار التوصيل حسب المنطقة لكل عميل.
- إدارة مناديب التوصيل (إضافة/تعديل/حذف).
- إدارة أنواع الطلبات.
- إنشاء طلبات مع بيانات المستلم وخيارات التحصيل.

## ملاحظات
- هذا المشروع مبني وفق هيكل Laravel القياسي (Models/Controllers/Views/Migrations).
- لا يعتمد على Node.js أو React ويستخدم Blade مع Bootstrap عبر CDN.

## الخطوات المقترحة للتشغيل
1. تثبيت Laravel محليًا عبر Composer.
2. نقل الملفات إلى مشروع Laravel جديد.
3. تشغيل المايجريشن لإعداد قاعدة البيانات.
4. تشغيل السيرفر المحلي باستخدام `php artisan serve`.

> ملاحظة: في بيئة بدون إنترنت قد تحتاج لإنشاء مشروع Laravel يدويًا ثم نسخ هذه الملفات إليه.
